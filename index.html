Events sites — unified stats

Aggregating GoatCounter analytics from events-hk, events-la, events-nrw, events-singa. Auto-refreshes when you switch range.

One-time setup needed. Sign up at goatcounter.com/signup and create one site with the code events (subdomain becomes events.goatcounter.com). Then in Settings → Site → Allow access to all (public statistics). Stats flow as soon as the next visit hits any of the four city pages.
